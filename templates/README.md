# RoadMap template

Place the generic RoadMap template here:

- **Generic_RoadMap.pptx** — required for "Download RoadMap PPTX". Must contain named placeholders (Selection Pane names) as used by `roadmap_pptx.py`:
  - **Slide 6:** text tokens `{{RETIREMENT_MONTHLY_DIFF}}`, `{{RETIREMENT_ANNUAL_DIFF}}`, `{{LIQUID_ASSETS_PRE}}`, `{{LIQUID_ASSETS_POST}}`
  - **Slide 8:** `TIMELINE_IMAGE`
  - **Slide 9:** `PRE_CASHFLOW_IMAGE` (top), `PRE_LIQUID_IMAGE` (bottom)
  - **Slide 12:** `PRE_CASHFLOW_COMPARISON`, `POST_CASHFLOW_COMPARISON`
  - **Slide 13:** `PRE_LIQUID_COMPARISON`, `POST_LIQUID_COMPARISON`
  - **Slide 19:** `COMP_CHART_1` … `COMP_CHART_4`
  - **Slide 24:** `POST_ESTATE_IMAGE`
